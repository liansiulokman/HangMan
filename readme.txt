-----readme-----

To implement this game in Python, user first need to save the 4 attached 
text files, including rules.txt, WordLibBeginner.txt, WordLibMedium.txt, 
and WordLibExpert.txt, in the C drive. Then run HangMan.py as main.

The documentation on program design is included in the HangMan.py as docstring.

Hope you enjoy the game!
